NexusUI

An in-game UI overlay framework for Starsector mods

Starsector 0.98a-RC7 v0.9.0-beta MIT License

NexusUI adds a floating overlay button to the Starsector campaign screen. Click it to open a tabbed panel where mods can register their own interactive pages — dashboards, cheat panels, colony managers, profilers, and more.

Download Latest Release

Features

  • Floating draggable overlay button on the campaign screen
  • Tabbed panel system with colored mod indicators
  • Simple NexusPage API for mods to register their own panels
  • Thread-safe game command queue for Swing-to-game-thread communication
  • Embedded REST API exposing game data as JSON
  • Themed drawing utilities for consistent Swing panel styling
